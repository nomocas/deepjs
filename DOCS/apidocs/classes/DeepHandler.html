treat

(
  • treatment
)
DeepHandler

Defined in deep.js:1455

apply a 'treatments' on chain entries (each entry will be the context of the treatment). (renderables are treatments - see doc and example).

a treaments is an object that contains : { what : (optional) object|retrievable_string a value to inject in 'how' function. if no 'what' is provided : inject the entry (the context) in 'how' how : a function to treat 'what' (simple function that have a single argument (what) and return its result), where: (optional) a function to send the results somewhere, return the descriptor of the sended ressource (see deep-ui : dom.apendTo for example), done : a callback function that will be called on treated entry (the context of the done) when treatment succeed fail : a callback function that will be called on treated entry (the context of the fail) when treatment failed }

if entry contain a 'treat' function : will be called and provided treatment will be passed as argument

Keep previous entries (maybe modified by treatment)

Chain Success Injection : the treatments results Chain Error Injection : the treatments errors

Parameters:

  • treatment Object

Returns:
